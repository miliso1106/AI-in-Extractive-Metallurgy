# Demo Datasets (Optimal, Warning, Alert)

These CSVs are simulated for classroom demo. Each folder contains 10 files with clearly named processes.

## Data source for structure
- Process names and base operating parameters are taken from `src/data/processData.json`.
- Recovery and efficiency values are adjusted to enforce the folder status:
  - Optimal: higher recovery/efficiency
  - Warning: mid recovery/efficiency
  - Alert: lower recovery/efficiency

If you want fully sourced industrial values for every field, tell me which references to use and I will replace the simulated values.
